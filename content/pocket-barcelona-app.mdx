### Description

An app for tourists and expats visiting and living in Barcelona

### More Details

Pocket Barcelona features a comprehensive directory, itinerary planner, going-out tips, an expat blog, a map for handy navigation and much more! The project is funded and run independently by local expats based in Barcelona.
