### Description

I completed a 2-day hackathon project for Hostshare for their candidate frontend developer test. The brief was to build 3 main pages from the Airbnb website (home, search results and property listing pages) but not feature complete

### More Details

The project was to demonstrate how I would go about building a similar copy, using Hostshare branding and assets. The data feed and framework (NextJS/Tailwind) were prescribed and part of the brief.
