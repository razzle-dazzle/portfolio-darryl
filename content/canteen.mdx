A lunch delivery service, based in Barcelona, Spain

### More Details

The website uses the Shopify Storefront API to handle product listings and checkout with a customized NextJS app to handle the UI. I also used Material Design for the UI framework. The site also has an orders dashboard which uses the Admin API, written in NextJS/Tailwind.
