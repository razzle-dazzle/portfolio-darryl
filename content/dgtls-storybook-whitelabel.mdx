### Description

A Storybook front-end boilerplate kit for bootstrapping new Storybook projects

### More Details

The core frontend team in DGTLS Barcelona developed a whitelabel Storybook repo, containing an essential UI component library, to speed up future Storybook projects. This later became a Lerna mono-repo, supporting React, Angular and Twig rendered UI components.
